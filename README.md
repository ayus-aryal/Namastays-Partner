# Namastays Partner

Namastays Partner is an Android application built for property owners and service providers to seamlessly register and manage their offerings on the Namastays platform. The app was created to simplify partner onboarding, event management, and secure interaction with users, reducing manual work and improving efficiency.  

## Purpose

The project is designed to:  
- Provide a **structured multi-step registration process** for partners, including OTP verification for security.  
- Enable partners to **manage their events and offerings** directly from the app.  
- Offer **location-based services** to autofill addresses and streamline data entry.  
- Ensure **secure authentication and data storage** using modern backend technologies.  

## Technology Stack

- **Language:** Kotlin  
- **UI Framework:** Jetpack Compose  
- **Backend:** Spring Boot
  
This app aims to provide a professional, secure, and user-friendly interface for partners, bridging the gap between property owners and the Namastays platform efficiently.
